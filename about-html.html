1. What is HTML?

HTML (HyperText Markup Language) provides the structure of web pages.

Why? Browsers need structure to display content correctly.

2. HTML Document Structure

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  Hello World
</body>
</html>

The head contains metadata; the body contains visible content.

3. Text Formatting

<h1>Heading</h1>
<p>Paragraph</p>
<strong>Bold</strong>
<em>Italic</em>

4. Lists

<ul>
  <li>HTML</li>
</ul>

<ol>
  <li>Step 1</li>
</ol>

5. Links & Navigation

<a href="https://example.com">Visit</a>

Links connect pages and resources.

6. Tables

<table>
<tr><th>Name</th><th>Age</th></tr>
<tr><td>Alex</td><td>25</td></tr>
</table>

7. Images, Audio & Video

<img src="image.jpg" alt="Description">

<audio controls>
 <source src="audio.mp3">
</audio>

<video controls width="300">
 <source src="video.mp4">
</video>

8. Forms and Input Types

<form>
<input type="text">
<input type="email">
<input type="password">
<input type="date">
<button>Submit</button>
</form>

Forms collect user data.

9. Semantic HTML

<header></header>
<nav></nav>
<main></main>
<article></article>
<section></section>
<footer></footer>
Semantic tags improve accessibility and SEO.

10. Accessibility Basics

  • Use alt text for images.
  • Use labels for form fields.
  • Maintain proper heading order.

Interactive Quiz

Q: Which tag contains visible content?
A: <body>

11. Best Practices

  • Use semantic tags.
  • Indent code properly.
  • Add alt attributes.
  • Validate your HTML.
  • Keep code readable.
Common Mistake: Using too many div elements instead of semantic tags.
