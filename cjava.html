1. Introduction to Java

Java is a platform-independent, object-oriented programming language used in enterprise software, Android apps, web applications, cloud systems, and more.

Why Java? Write Once, Run Anywhere (WORA).

2. Java Program Structure

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}

Output: Hello Java

Every Java application starts from the main() method. The JVM executes code from this entry point.

3. Data Types and Variables

int age = 25;
double salary = 50000.75;
char grade = 'A';
boolean active = true;
String name = "Avinash";

Variables store data in memory. Java is strongly typed, so each variable requires a datatype.

5. Control Flow

If-Else

int marks = 80;

if(marks >= 50){
    System.out.println("Pass");
}else{
    System.out.println("Fail");
}

For Loop

for(int i=1;i<=5;i++){
    System.out.println(i);
}
Common Mistake: Infinite loops occur when loop conditions never become false.

6. Methods

public static int add(int a,int b){
    return a+b;
}

Methods improve code reusability and modularity.

7. Object-Oriented Programming

Class and Object

class Student{
   String name;
   void display(){
      System.out.println(name);
   }
}

Encapsulation

class Employee{
   private int salary;

   public void setSalary(int salary){
      this.salary = salary;
   }

   public int getSalary(){
      return salary;
   }
}

Inheritance

class Animal{
   void sound(){
      System.out.println("Animal Sound");
   }
}

class Dog extends Animal{
}

Polymorphism

class Animal{
   void sound(){
      System.out.println("Animal");
   }
}

class Dog extends Animal{
   void sound(){
      System.out.println("Bark");
   }
}
OOP helps build scalable and maintainable applications.

9. Exception Handling

try{
   int x = 10/0;
}
catch(Exception e){
   System.out.println(e.getMessage());
}
finally{
   System.out.println("Cleanup");
}

Exception handling prevents unexpected program termination.

10. File I/O Basics

FileWriter fw = new FileWriter("data.txt");
fw.write("Hello Java");
fw.close();

BufferedReader br =
 new BufferedReader(new FileReader("data.txt"));

System.out.println(br.readLine());

File I/O allows programs to persist data.

12. Best Practices

  • Use meaningful variable names.
  • Follow camelCase naming.
  • Handle exceptions properly.
  • Keep methods small and focused.
  • Write reusable code.
  • Comment complex logic.
